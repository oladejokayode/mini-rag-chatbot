# Mini RAG Chatbot

This is a simple Retrieval-Augmented Generation (RAG) chatbot built with Python.

This project aims to understand how AI systems can retrieve relevant information from local documents and generate more grounded responses.

## Features
- Loads local text documents
- Converts them into embeddings
- Stores them using FAISS
- Retrieves relevant context for user queries

## Tools Used
- Python
- LangChain
- FAISS
- HuggingFace Transformers

## Status
Work in progress – improving response quality and structure
